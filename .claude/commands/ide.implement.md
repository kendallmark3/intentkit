# ide.implement

Implement the feature using the approved intent, context, plan, and tasks.

Instructions:
1. Read all files in the active intent workspace.
2. Implement tasks in order.
3. Use existing repo patterns from `repo-context.md`.
4. Keep changes minimal and reviewable.
5. Update task checkboxes as work is completed.
6. Stop and request human decision if the implementation requires architecture, schema, API contract, or security changes not documented in `architecture.md`.
7. End with recommended next command: `ide.verify`.
