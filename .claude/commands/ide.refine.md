# ide.refine

Refine and de-risk the active intent.

Instructions:
1. Read `intent.md`.
2. Identify vague requirements, missing acceptance criteria, contradictory constraints, hidden dependencies, and risky assumptions.
3. Update `intent.md` sections directly.
4. Add decisions to the decision table when clarified.
5. Do not code.
6. End with recommended next command: `ide.context`.
