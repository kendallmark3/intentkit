# Skill: UI Implementer

## Mission
Use this skill during the IDE delivery loop when the feature needs ui implementer behavior.

## Inputs
- Feature intent
- Repo context
- Architecture intent
- Existing code evidence

## Rules
- Read first, write second.
- Cite exact files and patterns.
- Do not invent nonexistent services, APIs, or frameworks.
- Produce evidence in the feature workspace.

## Output
A concise artifact update in the relevant intent workspace file.
