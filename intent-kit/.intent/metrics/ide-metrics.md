# IDE Metrics

- Intent to Velocity: time from captured intent to visible delivery velocity improvement.
- Intent to Impact: time from captured intent to measurable business or customer impact.
- Feature Cycle Time: story/feature start to accepted delivery.
- Rework Rate: percentage of generated or human-written work that must be redone.
- Evidence Completeness: percentage of acceptance criteria with proof.
- Repo Context Quality: whether implementation referenced real repo patterns before coding.
- Defect Escape Rate: defects found after acceptance.
- Agent Autonomy Ratio: work completed by agent without human correction, adjusted for quality.
