# ide.capture

Capture the feature intent before coding.

Instructions:
1. Read `.intent/memory/principles.md` and `.intent/memory/definition-of-done.md`.
2. Find the current feature workspace under `intents/` or ask which one if multiple are active.
3. Update `intent.md` with business intent, users, scope, inputs, acceptance intent, assumptions, questions, and decisions.
4. Do not implement code.
5. End with the top unresolved questions and recommended next command: `ide.refine`.
