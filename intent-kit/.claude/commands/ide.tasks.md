# ide.tasks

Generate ordered implementation tasks.

Instructions:
1. Read `intent.md`, `repo-context.md`, `architecture.md`, and `plan.md`.
2. Update `tasks.md` with a dependency-aware list of small, reviewable tasks.
3. Ensure every task traces to acceptance intent or technical risk.
4. Do not code.
5. End with recommended next command: `ide.implement`.
